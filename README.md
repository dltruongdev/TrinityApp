# TrinityApp

## 1. Requirements and Setups
- Go 1.23
- [Go Migrate](https://github.com/golang-migrate/migrate/tree/master/cmd/migrate)
- Make for Makefile
- Docker
- Steps:
  - Clone this repo
  - Run Make commands with orders (make pullpostgres, make postgres, make migrate-up)
  - Start the application
